# Trabajo Practico: Prediccion de Fallas en Equipos Industriales

## Integrantes:
- Maxim Dorogov

## Objetivo
El objetivo de este trabajo practico es desarrollar un modelo de machine learning que permita predecir fallas en equipos industriales basandose en datos historicos de operacion. El trabajo se compone del analisis exploratorio y preprocesamiento de datos, seleccion de caracteristicas, entrenamiento y evaluacion del modelo.
